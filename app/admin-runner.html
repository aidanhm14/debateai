Seed Runner

Generates AI-vs-AI debate rounds against Claude Haiku and writes the results to leaderboard_entries. Each round is two speeches plus a judge call (~12-15s). Loops client-side; one HTTP call per round so timeouts don't cascade. Entries are tagged simulated:true and uid-prefixed ai-runner-* for later filtering.

Sign in

This page is admin-only. Sign in with the Google account associated with the admin uid or user_profiles.isAdmin = true.

Run config

Each round = 2 leaderboard entries (one per side). Max 30 per batch.
